# AGEV-Prototype

Autonomous Ground Electric Vehicle (AGEV) built using ESP32, Ultrasonic sensors and Motor Drivers for obstacle detection and autonomous navigation.

## Overview
This project implements a 4WD autonomous electric ground vehicle capable of:
- Obstacle detection
- Autonomous navigation
- Motor control using PWM
- Wi-Fi based telemetry (optional cloud integration)

Developed during internship at Pragyatmika (Embedded Systems & IoT Division).

## Hardware
- ESP32
- L298N Motor Driver
- 4 DC Motors
- Ultrasonic Sensor (HC-SR04)
- LiDAR (optional)
- GPS (optional)
- Battery pack

## Features
- Forward motion
- Obstacle detection
- Auto stop + turn
- Differential drive control
- PWM speed control
